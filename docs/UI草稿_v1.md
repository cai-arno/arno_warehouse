## 1. 设计方向说明

### 1.1 核心矛盾

| 矛盾 | 常见误区 | 正确思路 |
|------|----------|----------|
| 目标用户 vs 决策者 | 设计给孩子看 | 家长是付款方，孩子是使用者，两者都要讨好 |
| 内容深度 vs 学习动力 | 堆知识点越全越好 | 用成就系统+游戏化驱动持续使用 |
| 功能全面 vs 界面简洁 | 第一版做全所有功能 | 聚焦「提分」这一个核心动作 |

### 1.2 UI 定位

```
风格关键词：专业但不冰冷、有趣但不幼稚、像学霸笔记而不是教辅书
主色调：学院蓝/知识绿（忌太红，像题库广告）
字体：宋体（正文）+ 活泼字体（标题）
目标用户：初中生（12-15岁）优先
```

### 1.3 核心页面优先级

1. **首页**（第一屏定生死）
2. **题库练习页**（核心提分场景）
3. **错题本页**（留存关键页）
4. **学习成就页**（激励系统）
5. **能力诊断页**（差异化亮点）
6. **初中专区-学科选择页**（内容入口）
