## 一、背景与目标

### 1.1 问题陈述

当前短视频工厂使用**通用模板**，无法满足以下垂直场景的深度需求：

| 场景 | 通用模板痛点 |
|------|------------|
| 电商种草 | 缺乏"产品卖点+价格钩子+购买引导"的完整链路 |
| 门店引流 | 缺乏"门店氛围+地址引导+限时优惠"的本地化元素 |
| 知识科普 | 缺乏"悬念开场+信息递进+总结行动号召"的结构化框架 |
| 活动促销 | 缺乏"倒计时+稀缺感+多重优惠"的促销文案节奏 |
| 节日热点 | 缺乏时效性模板和热点借势文案风格 |

### 1.2 目标

构建一套**行业垂直模板库**，支持：
- 按场景（5大类）和行业（10+子行业）快速筛选
- 变量占位符实现"一模板多内容"
- 模板预览、编辑、分类管理
- 设计师入驻生态（后续扩展）

#### 2.2.4 变量替换引擎

**功能描述：** 将模板中的 `{{variable_key}}` 占位符替换为实际内容。

**技术设计：**

```python
class TemplateEngine:
    """变量替换引擎"""

    def render_script(self, template: ScriptTemplate, variables: dict) -> str:
        """渲染脚本模板"""
        content = template.script.model_dump_json()
        for key, value in variables.items():
            content = content.replace(f"{{{{{key}}}}}", str(value))
        return content

    def validate_variables(self, template: ScriptTemplate, variables: dict) -> ValidationResult:
        """校验必填变量是否填充"""
        errors = []
        for var in template.variables:
            if var.required and var.key not in variables:
                errors.append(f"缺少必填变量: {var.label}")
            if var.type == "number" and var.key in variables:
                if not variables[var.key].isdigit():
                    errors.append(f"{var.label} 必须为数字")
        return ValidationResult(valid=len(errors) == 0, errors=errors)

    def get_missing_variables(self, template: ScriptTemplate, variables: dict) -> list[str]:
        """获取未填充的必填变量列表"""
        return [
            var.label for var in template.variables
            if var.required and var.key not in variables
        ]
```

## 三、技术方案

### 3.1 数据库设计

```sql
-- 模板分类表
CREATE TABLE template_categories (
    id VARCHAR(50) PRIMARY KEY,
    name VARCHAR(100) NOT NULL,
    parent_id VARCHAR(50) REFERENCES template_categories(id),
    level INT NOT NULL CHECK (level IN (1, 2)),
    icon VARCHAR(20),
    sort_order INT DEFAULT 0,
    is_active BOOLEAN DEFAULT true,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 脚本模板表
CREATE TABLE script_templates (
    id VARCHAR(50) PRIMARY KEY,
    name VARCHAR(200) NOT NULL,
    category_id VARCHAR(50) REFERENCES template_categories(id),
    hook TEXT NOT NULL,
    body JSONB NOT NULL,           -- ["第一句", "第二句", ...]
    cta TEXT NOT NULL,
    variables JSONB NOT NULL,      -- 变量定义列表
    estimated_duration INT,         -- 秒
    style_tags JSONB,              -- ["真实感", "闺蜜风"]
    score_weights JSONB,           -- 各维度权重
    is_active BOOLEAN DEFAULT true,
    use_count INT DEFAULT 0,
    rating DECIMAL(3,2),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
    updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 视频素材模板表
CREATE TABLE video_templates (
    id VARCHAR(50) PRIMARY KEY,
    script_template_id VARCHAR(50) REFERENCES script_templates(id),
    name VARCHAR(200),
    structure JSONB NOT NULL,      -- 片段结构
    recommended_bgm_tags JSONB,
    aspect_ratio VARCHAR(10) DEFAULT '9:16',
    platform_configs JSONB,        -- 各平台适配配置
    is_active BOOLEAN DEFAULT true,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 配音风格模板表
CREATE TABLE voice_templates (
    id VARCHAR(50) PRIMARY KEY,
    name VARCHAR(200) NOT NULL,
    category_ids JSONB,            -- 适用的分类列表
    tts_config JSONB NOT NULL,     -- TTS配置
    style_guide JSONB NOT NULL,    -- 风格指南
    is_active BOOLEAN DEFAULT true,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 模板标签表
CREATE TABLE template_tags (
    id SERIAL PRIMARY KEY,
    name VARCHAR(50) UNIQUE NOT NULL,
    color VARCHAR(20),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 模板-标签关联表
CREATE TABLE template_tag_relations (
    template_type VARCHAR(20),     -- 'script' / 'video' / 'voice'
    template_id VARCHAR(50),
    tag_id INT REFERENCES template_tags(id),
    PRIMARY KEY (template_type, template_id, tag_id)
);

-- 模板使用记录（统计用）
CREATE TABLE template_usage_logs (
    id SERIAL PRIMARY KEY,
    template_type VARCHAR(20) NOT NULL,
    template_id VARCHAR(50) NOT NULL,
    user_id VARCHAR(50) NOT NULL,
    project_id VARCHAR(50),
    used_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

## 六、风险与应对

| 风险 | 等级 | 应对措施 |
|------|------|---------|
