# 伴学网 - 项目规格说明书

## 1. 项目概述

**项目名称：** 伴学网  
**项目代号：** xuefeng-academy  
**核心理念：** 专注考试提分，让每个学生都能有好成绩、好学校、上大学不踩坑  
**项目负责人：** 枢机（直接向老大汇报）  
**数据来源：** 当当网、清华大学出版社、孔夫子旧书网等公开信息  
**状态：** 进行中

## 3. 产品定位

### 3.1 使命
- 帮助学生考试提分
- 培养终身学习习惯
- 让学生少走弯路

### 3.2 用户画像
- **初中生**：12-15岁，需要培养学习兴趣、掌握高效学习方法
- **高中生**：15-18岁，坚定学习信念、应对高考压力
- **大学生**：18-22岁（暂停）

### 4.8 题库系统（核心！必须入数据库）

#### 4.8.1 题库设计原则
- **每题必标出处**：中考真题/高考真题/模拟题/原创题
- **每题必入数据库**：PostgreSQL存储
- **每题必属知识点**：与章节知识点关联

#### 4.8.2 题库数据库结构

```
┌─────────────────────────────────────────────────────┐
│  题目主表 questions                                   │
├─────────────────────────────────────────────────────┤
│  id              主键                               │
│  content         题目内容                             │
│  answer          正确答案                             │
│  analysis        详细解析                            │
│  difficulty      难度(1-5星)                         │
│  question_type   题型(选择/填空/解答)                │
│  source          出处完整标注                        │
│  source_year     年份                               │
│  source_type     来源类型(中考/高考/模拟/原创)       │
│  score           分值                               │
│  chapter_id      所属章节ID                          │
│  created_at      创建时间                            │
└─────────────────────────────────────────────────────┘
         │
         ├── 题目选项表 question_options (选择题用)
         │
         ├── 知识点关联表 question_tags
         │
         └── 用户答题记录 user_answers
```

#### 7.5.4 评分规则配置

```sql
-- 题目评分规则表
CREATE TABLE scoring_rules (
    id SERIAL PRIMARY KEY,
    question_id INTEGER REFERENCES questions(id),
    scoring_type VARCHAR(20),          -- exact_match/keyword_coverage/step_based
    correct_answer TEXT NOT NULL,       -- 标准答案
    answer_pattern JSONB,             -- 答案模式（如填空题多个可接受答案）
    keyword_weights JSONB,             -- 关键词权重（用于主观题）
    step_scores JSONB,                -- 步骤分配置
    partial_credit BOOLEAN DEFAULT TRUE,  -- 是否给部分分
    created_at TIMESTAMP DEFAULT NOW()
);

-- 答题记录（含评分详情）
CREATE TABLE practice_records (
    id SERIAL PRIMARY KEY,
    user_id INTEGER REFERENCES users(id),
    question_id INTEGER REFERENCES questions(id),
    user_answer TEXT NOT NULL,
    correct_answer TEXT NOT NULL,      -- 评分时记录标准答案
    score DECIMAL(5,2),                -- 得分
    max_score DECIMAL(5,2),            -- 满分
    scoring_type VARCHAR(20),          -- 评分方式
    answer_analysis JSONB,              -- 详细评分分析
    is_correct BOOLEAN,
    submitted_at TIMESTAMP DEFAULT NOW()
);
```
